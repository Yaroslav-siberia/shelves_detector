'''
array с array-ями
вложенный array это полка. соответственно полки упорядочены сверху вниз
Внутри полки - объекты, они упорядочены слева на право.
описание объекта
detection = {"category": "", "group_box": obj, "obj_boxes": [obj]}
структура детектируемого объекта
category:str - название категории, заполняется на этапе кластеризации
group_box: BigBBox - большая рамка вокруг нескольких объектов одной категории. пока товар один и не кластеризован рамка
соответствует рамке товара
obj_boxes: list(BBox) - список с рамками, каждая из которых вокруг отдельного объекта. пока объекты
не кластеризованны то оставляем там один объект
'''
